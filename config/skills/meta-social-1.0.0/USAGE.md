# How to ask (Meta Social skill)

- "Post to my Facebook Page: Hello everyone"
- "Post this to Facebook: [message]"
- "Post to Instagram: [public image URL] with caption [text]"
- "Publish this image to Instagram: [image URL]"

You need:
- **META_ACCESS_TOKEN** (or `meta_access_token` in this skill's config.yml) — Page access token from [developers.facebook.com](https://developers.facebook.com)
- **Page ID** — numeric Facebook Page ID (for both Facebook and Instagram; for Instagram the Page must have an Instagram Business account linked)
