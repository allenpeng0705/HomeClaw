HomeClaw Companion App

The best way to interact with HomeClaw. Available for Mac, Windows, iPhone, and Android.

The Companion app provides a beautiful, intuitive interface for chatting with HomeClaw. It connects to your Core and gives you full access to all features, including memory, plugins, and skills.

Setup the Companion App

Follow these steps to connect the Companion app to your HomeClaw Core:

Step 1: Install the App

  • Download and install the Companion app for your platform
  • Launch the application

Step 2: Configure Core URL

Choose your connection method:

Option A: Local LAN Connection

If your device is on the same local network as HomeClaw Core:

  1. Find your Core's local IP address (e.g., 192.168.1.100)
  2. In Companion app Settings, enter: http://<local-ip>:9000
  3. Example: http://192.168.1.100:9000

Option B: Remote Connection via Tunnel

For iPhone, Android, or remote access, use a tunnel service:

  • Cloudflare Tunnel: Set up a tunnel to get a domain like https://your-subdomain.trycloudflare.com
  • Pinggy: Use Pinggy to create a tunnel with a custom domain
  • Ngrok: Use Ngrok to get a public URL like https://abc123.ngrok.io

Enter the tunnel URL in the Companion app as your Core URL.

Step 3: Enable API Key Authentication (Recommended)

Secure your Core connection with API key authentication:

Enable authentication in config/core.yml

auth_enable: true
auth_api_key: your-secure-api-key-here

After enabling authentication:

  1. Open Settings in the Companion app
  2. Enter the same API key you configured in core.yml
  3. Save the settings
Important: Always enable API key authentication when using remote access (tunneling). Without authentication, anyone with your tunnel URL can access your Core.

Step 4: Add Your User

Add your user in one of two ways:

  • Method 1: Edit config/user.yml and add your user under im:
  • Method 2: Use "Manage Core" feature in the app (if available)

Step 5: Connect and Chat

  1. Click "Connect" in the app
  2. Wait for connection to establish
  3. Start chatting with HomeClaw!

Remote Access

Access HomeClaw from anywhere using the Companion app:

QR Code Scan (Recommended)

Use the built-in Pinggy tunnel for easy remote access:

  1. Set pinggy.token in config/core.yml
  2. Start Core: python -m main start
  3. Core starts Pinggy tunnel and opens http://127.0.0.1:<port>/pinggy
  4. In Companion app: Settings → Scan QR to connect
  5. Scan the QR code from the /pinggy page
  6. App automatically saves Core URL and API key

Manual Configuration

Configure remote access manually using:

  • Tailscale: Use Tailscale IP as Core URL
  • Cloudflare Tunnel: Use tunnel URL as Core URL
  • SSH Tunnel: Forward port to Core over SSH
Security Tip: Always enable auth_enabled and auth_api_key in config/core.yml when using remote access to protect your Core.

Troubleshooting

Common issues and solutions:

Cannot Connect to Core

  • Make sure Core is running: python -m main start
  • Check Core URL is correct (local: http://127.0.0.1:9000 or LAN IP)
  • For remote access, verify your tunnel is running
  • Verify firewall settings allow the connection
  • Check if authentication is enabled and API key is correct

Connection Drops Frequently

  • Check network stability
  • Try using a different remote access method
  • Ensure Core is running on a stable connection

Messages Not Syncing

  • Check if both devices are connected to the same Core
  • Verify user configuration in config/user.yml
  • Check Core logs for errors
