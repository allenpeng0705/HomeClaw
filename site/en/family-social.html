Family Social Network

A private social network for your family, powered by HomeClaw.

Create a secure, AI-enhanced social network within your home. Each family member gets their own personalized space with AI friends, while staying connected with each other.

Overview

The Family Social Network transforms HomeClaw into a private social platform where family members can interact with AI assistants and each other, all within the security of your home network.

Key Features

  • Private & Secure: All data stays on your local network
  • Personalized Spaces: Each user has their own sandbox with isolated data
  • AI Friends: Pre-configured AI assistants with specialized capabilities
  • Family Connections: Users can add each other as friends
  • Companion App: Access the social network from any device

User Management

Each family member gets their own account with personalized settings and data isolation.

Creating Users

You can add family members in two ways:

Option 1: Using Portal (Recommended)

The easiest way to manage users is through the Portal web interface:

  1. Start Portal: python -m main portal
  2. Open browser to http://127.0.0.1:18472
  3. Navigate to User Management section
  4. Click "Add User" and fill in the details
  5. Save changes

Option 2: Editing user.yml

Alternatively, add family members by editing config/user.yml:

User configuration example

users:
  - id: dad
    name: Dad
    username: dad
    password: your-secure-password
    email: []
    im: []
    phone: []
    permissions: []
  
  - id: mom
    name: Mom
    username: mom
    password: your-secure-password
    email: []
    im: []
    phone: []
    permissions: []

User Sandbox

Each user has their own isolated environment:

  • Personal Folder: Private file storage in sandbox/<user_id>/
  • Session Data: Separate conversation history and preferences
  • AI Friends: Each user can customize their AI friend list
  • Settings: Individual configuration per user

User Fields

  • id: Unique identifier for the user
  • name: Display name
  • username: Login username
  • password: User's password
  • email: Email addresses for the user
  • im: IM channel identities (e.g., 'matrix:@user:matrix.org', 'telegram:123456')
  • phone: Phone numbers for SMS
  • permissions: Channel permissions (empty array [] allows all)

Custom AI Friends

Users can create their own AI friends with custom personalities and capabilities:

Option 1: Using Portal (Recommended)

Add AI friends through the Portal web interface:

  1. Open Portal at http://127.0.0.1:18472
  2. Navigate to User Management
  3. Select the user to add friends for
  4. Click "Add Friend"
  5. Choose preset friend (Reminder, Finder, Cursor, ClaudeCode) or create custom
  6. Configure personality and settings
  7. Save changes

Option 2: Editing user.yml

Alternatively, add AI friends by editing config/user.yml:

Add custom AI friend in config/user.yml

users:
  - id: alice
    name: Alice
    username: alice
    password: your-password
    email: []
    im: []
    phone: []
    permissions: []
    friends:
      - name: HomeClaw
      - name: Chef
        relation: friend
        who:
          gender: 'neutral'
          roles: ['cooking_assistant']
          personalities: [friendly, enthusiastic, knowledgeable]
          language: en
          response_length: medium

Remote Access

Access your Family Social Network from anywhere using secure tunneling solutions.

Remote Access Options

Connect to your HomeClaw from outside your home network:

Pinggy

Create secure tunnels with custom domains:

  • Set up in config/core.yml with your Pinggy token
  • Get a public URL like https://xxx.pinggy.io
  • Easy QR code scanning for mobile setup

Cloudflare Tunnel

Free, secure tunneling with custom subdomains:

  • Install cloudflared on your HomeClaw server
  • Create a tunnel to get https://your-name.trycloudflare.com
  • No account required for basic usage

Ngrok

Popular tunneling service for developers:

  • Get instant public URLs like https://abc123.ngrok.io
  • Free tier available
  • Easy to set up and use

Tailscale

Zero-config VPN for secure mesh networking:

  • Create a private network between devices
  • Use Tailscale IP to connect (e.g., http://100.x.x.x:9000)
  • Works like you're on the same local network
Security Reminder: Always enable auth_enable: true and set a strong auth_api_key in config/core.yml when using remote access. This ensures only authorized users can connect to your HomeClaw.

Setup Guide

Follow these steps to set up your Family Social Network:

Step 1: Configure Users

Edit config/user.yml to add family members:

Example user.yml

users:
  - id: dad
    name: Dad
    username: dad
    password: secure_password_1
    email: []
    im: []
    phone: []
    permissions: []
    friends:
      - name: HomeClaw
      - name: Reminder
        preset: reminder
      - name: Finder
        preset: finder
      - name: Cursor
        preset: cursor
      - name: ClaudeCode
        preset: claudecode
  
  - id: mom
    name: Mom
    username: mom
    password: secure_password_2
    email: []
    im: []
    phone: []
    permissions: []
    friends:
      - name: HomeClaw
      - name: Reminder
        preset: reminder

Step 2: Start Core

Start HomeClaw Core

python -m main start

Step 3: Install Companion App

Each family member installs the Companion app:

  • Download from GitHub Releases
  • iPhone users: Coming soon to App Store

Step 4: Connect to Core

Each user connects their Companion app:

  1. Open Companion app Settings
  2. Enter Core URL (local IP or tunnel URL)
  3. Enter API key (if authentication is enabled)
  4. Save settings and connect

Step 5: Login and Chat

  1. Each user logs in with their username and password
  2. Start chatting with AI friends
  3. Add family members as friends

Privacy & Security

The Family Social Network is designed with privacy and security as top priorities.

Data Isolation

  • Sandbox Folders: Each user's files are stored in separate directories
  • Session Separation: Conversation history is isolated per user
  • No Data Sharing: AI friends don't share information between users
  • Local Storage: All data stays on your HomeClaw server

Authentication

  • User Passwords: Each user has their own secure password
  • API Key: Optional API key for app connections
  • Role-Based Access: Admin and user roles for different permissions

Network Security

  • Local Network: Default access limited to local network
  • Remote Access: Optional tunneling for secure remote connections
  • No Cloud: No data sent to external servers
Your Data, Your Control: Unlike public social networks, your family's conversations and data never leave your home. You have complete control over your information.
