HomeClaw Portal

Your local web interface for managing and configuring HomeClaw.

The Portal provides a convenient web interface for configuring your HomeClaw instance. Use it to manage models, configure settings, and monitor your system — all from your browser.

On this page

  • Access Portal

Access Portal

The Portal runs locally and is accessible from your web browser:

Start the Portal

Start Portal

python -m main portal

Access the Portal

Once started, open your browser and navigate to:

Portal URL

http://127.0.0.1:18472
Note: The Portal is designed for local access only. It provides a convenient interface for managing your HomeClaw configuration without editing YAML files directly.

Portal Features

The Portal offers a comprehensive set of management features:

Model Configuration

Configure your local and cloud models through an intuitive interface:

  • Add and configure local models (GGUF files)
  • Set up cloud model providers (OpenAI, DeepSeek, etc.)
  • Configure Mix Mode for optimal performance
  • Set default models for main LLM and embedding

Settings Management

Manage all HomeClaw settings from the Portal:

  • Edit core.yml settings
  • Configure user preferences
  • Manage LLM parameters
  • Set up authentication

Plugin Management

Manage your plugins directly from the Portal:

  • View installed plugins
  • Enable or disable plugins
  • Configure plugin settings
  • Monitor plugin status

Channel Configuration

Configure communication channels:

  • Set up Telegram bot
  • Configure Discord integration
  • Manage WebChat settings
  • Test channel connections

System Monitoring

Monitor your HomeClaw instance:

  • View system status
  • Check model availability
  • Monitor resource usage
  • View recent logs

Management Tools

Powerful management capabilities accessible from the Portal:

Configuration Files

Edit configuration files through the web interface:

  • core.yml — Main Core configuration
  • llm.yml — Model and LLM settings
  • user.yml — User profiles and preferences
  • plugins/ — Plugin configurations

Service Control

Control HomeClaw services from the Portal:

  • Start and stop Core
  • Restart services
  • Check service status
  • View service logs

Data Management

Manage your HomeClaw data:

  • View conversation history
  • Export and import configurations
  • Backup settings
  • Clear cached data

Authentication Setup

Configure security settings:

  • Enable/disable API key authentication
  • Set or change API key
  • Configure allowed hosts
  • Manage access permissions

Enable authentication in core.yml

auth_enable: true
auth_api_key: your-secure-api-key-here

Troubleshooting

Common Portal issues and solutions:

Cannot Access Portal

  • Ensure Portal is running: python -m main portal
  • Check if port 18472 is available
  • Verify the URL is correct: http://127.0.0.1:18472
  • Check if another process is using the port

Portal Not Starting

  • Check Python version (requires 3.11+)
  • Verify all dependencies are installed
  • Check Portal logs for errors
  • Try a different port: python -m main portal --port 18473

Configuration Changes Not Saving

  • Check file permissions in config/ directory
  • Verify YAML syntax is correct
  • Check for conflicting settings
  • Restart Portal after making changes
