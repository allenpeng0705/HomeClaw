Remote Access

Connect to your HomeClaw from anywhere in the world.

Access your Family Social Network, chat with AI friends, and manage your HomeClaw remotely using secure tunneling solutions. Perfect for iPhone, Android, and remote desktop access.

Overview

When you're away from home, you can still access your HomeClaw and Family Social Network using various tunneling solutions. These create secure connections from your devices to your HomeClaw server.

Why Remote Access?

  • iPhone/Android: Use Companion app on mobile devices from anywhere
  • Family Social Network: Stay connected with family members remotely
  • AI Friends: Chat with your AI assistants on the go
  • Remote Work: Access HomeClaw while traveling or at work

Connection Methods

  • Local LAN: Use local IP when on the same network (e.g., http://192.168.1.100:9000)
  • Tunnel Services: Use when away from home (Pinggy, Cloudflare, Ngrok, etc.)
Important: Always enable API key authentication (auth_enable: true and auth_api_key) when using remote access. This prevents unauthorized access to your HomeClaw.

Pinggy (Built-in)

Pinggy is the easiest way to set up remote access. HomeClaw has built-in support for Pinggy with QR code scanning.

Setup Steps

  1. Get a Pinggy token from pinggy.io
  2. Add the token to your config/core.yml:

Configure Pinggy in core.yml

pinggy:
  token: your-pinggy-token-here
  enabled: true
  1. Start HomeClaw Core: python -m main start
  2. Core will automatically start the Pinggy tunnel
  3. Open the Pinggy URL (e.g., http://127.0.0.1:9000/pinggy) in your browser
  4. Scan the QR code with your Companion app

QR Code Connection

The easiest way to connect your Companion app:

  1. On your HomeClaw server, open http://127.0.0.1:9000/pinggy
  2. A QR code will be displayed
  3. In Companion app: Settings → Scan QR to Connect
  4. Scan the QR code
  5. App automatically configures the URL and API key

Advantages

  • Built into HomeClaw - no additional software needed
  • QR code scanning for easy mobile setup
  • Custom domains available
  • Secure encrypted connections

Cloudflare Tunnel

Cloudflare Tunnel provides free, secure tunneling with custom subdomains. No account required for basic usage.

Setup Steps

  1. Install cloudflared on your HomeClaw server:

Install cloudflared (Mac/Linux)

brew install cloudflared

Install cloudflared (Windows)

winget install --id Cloudflare.cloudflared
  1. Create a tunnel:

Create Cloudflare tunnel

cloudflared tunnel --url http://localhost:9000
  1. You'll get a URL like https://your-subdomain.trycloudflare.com
  2. Use this URL in your Companion app as the Core URL

Persistent Tunnel (Optional)

For a permanent tunnel with a custom subdomain:

  1. Create a Cloudflare account
  2. Authenticate cloudflared: cloudflared login
  3. Create a named tunnel: cloudflared tunnel create homeclaw
  4. Configure the tunnel in Cloudflare dashboard
  5. Run as a service for automatic startup

Advantages

  • Free tier with generous limits
  • No account required for quick tunnels
  • Custom subdomains with account
  • Highly reliable and fast

Ngrok

Ngrok is a popular tunneling service that provides instant public URLs for your local server.

Setup Steps

  1. Sign up at ngrok.com
  2. Download and install ngrok
  3. Authenticate with your token:

Authenticate ngrok

ngrok authtoken your-token-here
  1. Start the tunnel:

Start ngrok tunnel

ngrok http 9000
  1. You'll get a URL like https://abc123.ngrok.io
  2. Use this URL in your Companion app

Custom Domains (Paid)

With a paid ngrok plan, you can use custom domains:

Custom domain tunnel

ngrok http --domain=your-domain.ngrok.io 9000

Advantages

  • Easy to set up and use
  • Free tier available
  • Web interface for inspecting traffic
  • Custom domains with paid plans

Tailscale

Tailscale creates a secure mesh VPN network between your devices. It works like you're on the same local network, even when remote.

Setup Steps

  1. Sign up at tailscale.com
  2. Install Tailscale on your HomeClaw server:

Install Tailscale

# Mac
brew install tailscale

# Linux
curl -fsSL https://tailscale.com/install.sh | sh

# Windows
# Download from tailscale.com
  1. Start Tailscale and authenticate:

Start Tailscale

sudo tailscale up
  1. Get your HomeClaw server's Tailscale IP:

Check Tailscale IP

tailscale ip -4
  1. Install Tailscale on your mobile device or remote computer
  2. Connect to the same Tailscale network
  3. Use the Tailscale IP in Companion app: http://100.x.x.x:9000

Advantages

  • Works like a local network - no public URLs needed
  • Free for personal use (up to 20 devices)
  • Very secure - uses WireGuard encryption
  • No port forwarding required
  • Works across multiple devices seamlessly

SSH Tunnel

If you have a server with a public IP, you can use SSH tunneling to forward connections to your HomeClaw.

Setup Steps

  1. Have a server with SSH access and a public IP
  2. From your HomeClaw server, create an SSH tunnel:

Create SSH tunnel

ssh -R 9000:localhost:9000 user@your-server.com
  1. This forwards port 9000 from your server to HomeClaw
  2. Use your server's IP in Companion app: http://your-server.com:9000

Persistent SSH Tunnel

Use autossh for a persistent tunnel:

Install and use autossh

# Install autossh
brew install autossh  # Mac
sudo apt install autossh  # Linux

# Create persistent tunnel
autossh -M 0 -N -R 9000:localhost:9000 user@your-server.com

Advantages

  • No additional services needed if you have a server
  • Very secure (SSH encryption)
  • Full control over the connection
  • No third-party services required

Security

When exposing HomeClaw to the internet, security is critical.

Enable Authentication

Always enable API key authentication in config/core.yml:

Enable authentication

auth_enable: true
auth_api_key: your-very-secure-random-key-here

Security Best Practices

  • Use Strong API Keys: Generate a long, random API key
  • Rotate Keys Regularly: Change your API key periodically
  • HTTPS Only: Use HTTPS tunnels when possible (Cloudflare, Ngrok HTTPS)
  • Limit Access: Only share the tunnel URL with trusted family members
  • Monitor Connections: Check Core logs for unauthorized access attempts

Firewall Considerations

  • Tunnel services handle external access - no port forwarding needed
  • For Tailscale, ensure firewall allows Tailscale traffic
  • SSH tunnel requires SSH port (22) to be accessible
Warning: Never expose HomeClaw to the internet without authentication enabled. Without auth_enable: true, anyone with your tunnel URL can access your Core and data.

Connect Companion App

Once your tunnel is set up, connect your Companion app to access HomeClaw remotely.

Connection Steps

  1. Open Companion app on your device
  2. Go to Settings
  3. Enter the tunnel URL as Core URL:
  • Pinggy: https://xxx.pinggy.io
  • Cloudflare: https://your-name.trycloudflare.com
  • Ngrok: https://abc123.ngrok.io
  • Tailscale: http://100.x.x.x:9000
  • SSH: http://your-server.com:9000
  1. Enter your API key from config/core.yml
  2. Save settings
  3. Click Connect

Family Social Network Access

With remote access configured, family members can:

  • Chat with AI friends from anywhere
  • Message each other in the Family Social Network
  • Receive push notifications for new messages
  • Access files and reminders remotely
  • Use offline mode when connection is unstable

Multiple Devices

Each family member can connect multiple devices:

  • iPhone and iPad
  • Android phone and tablet
  • Mac and Windows computers
  • All devices stay in sync
