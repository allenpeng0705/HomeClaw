家庭社交网络

为您的家庭打造的私密社交网络，由 HomeClaw 提供支持。

在家中创建一个安全、AI 增强的社交网络。每个家庭成员都有自己的个性化空间，配备 AI 好友，同时彼此保持联系。

概述

家庭社交网络将 HomeClaw 转变为一个私密社交平台，家庭成员可以与 AI 助手和彼此互动，全部在家庭网络的安全范围内。

主要功能

  • 私密安全： 所有数据保留在本地网络中
  • 个性化空间： 每个用户都有自己的沙盒，数据相互隔离
  • AI 好友： 预配置的 AI 助手，具备专业功能
  • 家庭连接： 用户可以互相添加为好友
  • 伴侣应用： 从任何设备访问社交网络

用户管理

每个家庭成员都有自己的账户，具有个性化设置和数据隔离。

创建用户

您可以通过两种方式添加家庭成员：

选项 1：使用 Portal（推荐）

通过 Portal Web 界面管理用户是最简单的方式：

  1. 启动 Portal：python -m main portal
  2. 在浏览器中打开 http://127.0.0.1:18472
  3. 导航到用户管理部分
  4. 点击"添加用户"并填写详细信息
  5. 保存更改

选项 2：编辑 user.yml

或者，通过编辑 config/user.yml 添加家庭成员：

用户配置示例

users:
  - id: dad
    name: 爸爸
    username: dad
    password: your-secure-password
    email: []
    im: []
    phone: []
    permissions: []
  
  - id: mom
    name: 妈妈
    username: mom
    password: your-secure-password
    email: []
    im: []
    phone: []
    permissions: []

用户沙盒

每个用户都有自己的隔离环境：

  • 个人文件夹： 私人文件存储在 sandbox/<user_id>/
  • 会话数据： 独立的对话历史和偏好设置
  • AI 好友： 每个用户可以自定义自己的 AI 好友列表
  • 设置： 每个用户独立配置

用户字段

  • id： 用户的唯一标识符
  • name： 显示名称
  • username： 登录用户名
  • password： 用户密码
  • email： 用户的电子邮件地址
  • im： IM 频道身份（例如 'matrix:@user:matrix.org', 'telegram:123456'）
  • phone： 用于短信的电话号码
  • permissions： 频道权限（空数组 [] 允许所有）

AI 好友

每个用户都有一套个性化的 AI 好友，每个好友都有独特的能力。

默认 AI 好友

每个用户自动拥有以下 AI 好友：

HomeClaw（主 AI）

您的主要 AI 助手，用于一般对话、问答和任务。

  • 通用知识和帮助
  • 协助日常任务和提醒
  • 回答任何问题
  • 访问所有配置的技能和插件

Reminder（提醒助手）

专门管理提醒和日程安排。

  • 设置和管理提醒
  • 创建待办事项清单
  • 安排通知
  • 跟踪截止日期和活动

File（文件助手）

您的个人文件管理助手。

  • 搜索和整理文件
  • 总结文档内容
  • 从文件中提取信息
  • 管理您的沙盒文件夹

Cursor（代码助手）

代码和开发助手，可以帮助您远程编码。

  • 在您的项目上编写和审查代码
  • 远程调试编程问题
  • 解释技术概念
  • 协助软件项目并返回结果
  • 访问和修改您沙盒中的文件

ClaudeCode（高级代码助手）

用于复杂开发任务的高级编码和技术助手。

  • 复杂代码分析和重构
  • 架构建议
  • 最佳实践指导
  • 技术文档帮助
  • 远程编码协助，结果返回给您

Note（笔记助手）

您的个人笔记和知识助手。

  • 创建和整理笔记
  • 总结信息
  • 建立个人知识库
  • 快速信息检索

自定义 AI 好友

用户可以创建自己的 AI 好友，自定义个性和能力：

选项 1：使用 Portal（推荐）

通过 Portal Web 界面添加 AI 好友：

  1. 在浏览器中打开 Portal http://127.0.0.1:18472
  2. 导航到用户管理
  3. 选择要添加好友的用户
  4. 点击"添加好友"
  5. 选择预设好友（Reminder、Finder、Cursor、ClaudeCode）或创建自定义好友
  6. 配置个性和设置
  7. 保存更改

选项 2：编辑 user.yml

或者，通过编辑 config/user.yml 添加 AI 好友：

在 config/user.yml 中添加自定义 AI 好友

users:
  - id: alice
    name: 爱丽丝
    username: alice
    password: your-password
    email: []
    im: []
    phone: []
    permissions: []
    friends:
      - name: HomeClaw
      - name: Chef
        relation: friend
        who:
          gender: 'neutral'
          roles: ['cooking_assistant']
          personalities: [friendly, enthusiastic, knowledgeable]
          language: zh
          response_length: medium

远程访问

使用安全的隧道解决方案从任何地方访问您的家庭社交网络。

远程访问选项

从家庭网络外部连接到您的 HomeClaw：

Pinggy

创建带有自定义域名的安全隧道：

  • 在 config/core.yml 中设置您的 Pinggy 令牌
  • 获取公共 URL，如 https://xxx.pinggy.io
  • 通过二维码扫描轻松设置移动设备

Cloudflare Tunnel

免费、安全的隧道，带有自定义子域名：

  • 在您的 HomeClaw 服务器上安装 cloudflared
  • 创建隧道以获取 https://your-name.trycloudflare.com
  • 基本使用无需账户

Ngrok

受开发者欢迎的隧道服务：

  • 获取即时公共 URL，如 https://abc123.ngrok.io
  • 提供免费套餐
  • 易于设置和使用

Tailscale

零配置 VPN，用于安全网状网络：

  • 在设备之间创建私有网络
  • 使用 Tailscale IP 连接（例如 http://100.x.x.x:9000）
  • 就像在同一个本地网络上一样工作
安全提醒： 使用远程访问时，请始终在 config/core.yml 中启用 auth_enable: true 并设置强 auth_api_key。这确保只有授权用户才能连接到您的 HomeClaw。

主要功能

家庭社交网络专为无缝家庭通信而设计。

推送通知

通过即时推送通知，永远不会错过任何消息：

  • 当家庭成员发送消息时收到通知
  • AI 好友回复触发通知
  • 每个用户可自定义通知偏好
  • 支持远程访问（隧道）

离线模式

即使没有互联网也能保持连接：

  • 离线时查看和阅读过去的对话
  • 撰写消息，恢复在线后同步
  • 所有数据缓存在您的设备本地
  • 连接恢复时自动同步

AI 好友对话

与 AI 好友聊天以获取帮助和协助：

  • 作业帮助： 孩子们可以向 AI 好友寻求学习帮助
  • 烹饪技巧： 从 AI 厨师那里获取食谱建议
  • 代码审查： 开发者可以从 Cursor 或 ClaudeCode 获得代码帮助
  • 笔记记录： 使用 Note AI 快速捕捉想法
  • 提醒： 使用 Reminder AI 设置个人提醒
  • 文件搜索： 让 File AI 查找文档

家庭连接

直接与家庭成员联系：

  • 添加家庭成员为好友
  • 发送私人消息
  • 共享文件和文档
  • 创建群聊

设置指南

按照以下步骤设置您的家庭社交网络：

步骤 1：配置用户

编辑 config/user.yml 添加家庭成员：

user.yml 示例

users:
  - id: dad
    name: 爸爸
    username: dad
    password: secure_password_1
    email: []
    im: []
    phone: []
    permissions: []
    friends:
      - name: HomeClaw
      - name: Reminder
        preset: reminder
      - name: Finder
        preset: finder
      - name: Cursor
        preset: cursor
      - name: ClaudeCode
        preset: claudecode
  
  - id: mom
    name: 妈妈
    username: mom
    password: secure_password_2
    email: []
    im: []
    phone: []
    permissions: []
    friends:
      - name: HomeClaw
      - name: Reminder
        preset: reminder

步骤 2：启动 Core

启动 HomeClaw Core

python -m main start

步骤 3：安装伴侣应用

每个家庭成员安装伴侣应用：

  • 从 GitHub Releases 下载
  • iPhone 用户：即将上架 App Store

步骤 4：连接到 Core

每个用户连接他们的伴侣应用：

  1. 打开伴侣应用设置
  2. 输入 Core URL（本地 IP 或隧道 URL）
  3. 输入 API 密钥（如果启用了认证）
  4. 保存设置并连接

步骤 5：登录和聊天

  1. 每个用户使用用户名和密码登录
  2. 开始与 AI 好友聊天
  3. 添加家庭成员为好友

隐私与安全

家庭社交网络将隐私和安全作为首要任务。

数据隔离

  • 沙盒文件夹： 每个用户的文件存储在单独的目录中
  • 会话分离： 每个用户的对话历史相互隔离
  • 无数据共享： AI 好友不会在用户之间共享信息
  • 本地存储： 所有数据保留在您的 HomeClaw 服务器上

认证

  • 用户密码： 每个用户都有自己的安全密码
  • API 密钥： 可选的 API 密钥用于应用连接
  • 基于角色的访问： 管理员和用户角色具有不同权限

网络安全

  • 本地网络： 默认访问仅限于本地网络
  • 远程访问： 可选的隧道用于安全远程连接
  • 无云端： 数据不会发送到外部服务器
您的数据，您掌控： 与公共社交网络不同，您家人的对话和数据永远不会离开您的家。您完全掌控自己的信息。
