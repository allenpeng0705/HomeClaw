如何安装 HomeClaw

在几分钟内让 HomeClaw 在您的机器上运行。支持 Mac、Windows 和 Linux。

HomeClaw 是一个在本地机器上运行的 AI 助手。您的数据由您自己掌控，为您提供完全的控制权和隐私保护。安装简单直接。

前置要求

在安装 HomeClaw 之前，请确保您具备以下条件：

  • Python 3.11 或更高版本 — 使用 python --version 或 python3 --version 检查
  • Git — 用于克隆仓库
  • pip — Python 包管理器（通常随 Python 一起安装）

注意：在 Windows 上，如果遇到 C++ 构建错误，请安装 Visual C++ Build Tools。

简单安装（推荐）

安装 HomeClaw 最简单的方法是使用提供的安装脚本。这些脚本会自动处理所有设置。

Mac & Linux

运行安装脚本

./install.sh

脚本将自动：

  • 克隆仓库（如果尚未克隆）
  • 安装 Python 依赖
  • 复制 llama.cpp 二进制分发文件
  • 下载所需的模型
  • 设置配置文件

Windows

对于 Windows，您可以使用 PowerShell 或命令提示符：

选项 1：PowerShell

.\install.ps1

选项 2：命令提示符

install.bat
脚本的作用： 安装脚本自动化整个设置过程，包括克隆代码、安装依赖、设置 llama.cpp 二进制文件和下载模型。这是大多数用户推荐的方法。

手动安装

如果您更喜欢不使用安装脚本手动安装 HomeClaw，请按照以下步骤操作：

步骤 1：克隆仓库

git clone https://github.com/allenpeng0705/HomeClaw.git
cd HomeClaw

步骤 2：安装 Python 依赖

pip install -r requirements.txt

步骤 3：复制 llama.cpp 二进制分发文件

下载并复制适合您平台的 llama.cpp 二进制文件：

  • Mac： 将二进制文件复制到适当的目录
  • Linux： 将二进制文件复制到适当的目录
  • Windows： 将二进制文件复制到适当的目录

步骤 4：下载模型

下载所需的 AI 模型。具体模型取决于您在 config/llm.yml 中的配置。

步骤 5：配置并运行

根据需要编辑配置文件，然后运行 HomeClaw：

python -m main start
注意： 手动安装需要更多的技术知识。除非您有特定的自定义设置要求，否则我们建议使用安装脚本。

验证安装

安装完成后，验证一切是否正常工作：

启动 HomeClaw

python -m main start

如果 HomeClaw 启动没有错误，并且您可以访问 Web 界面，恭喜！HomeClaw 已成功安装。

故障排除

常见问题和解决方案：

找不到 Python

确保 Python 已安装并添加到系统 PATH。在 Mac 上，您可能需要使用 python3 而不是 python。

找不到 pip

确保 pip 已安装。您可以使用以下命令安装：python -m ensurepip --upgrade

权限被拒绝

在 Mac/Linux 上，确保安装脚本具有执行权限：chmod +x install.sh

Windows 上出现构建错误

从 Microsoft 网站安装 Visual C++ Build Tools。

查看 GitHub issues 或 完整文档 以获取更详细的故障排除指南。
