前置要求

在安装 HomeClaw 之前，请确保您已具备：

Python 3.11+

运行 HomeClaw Core 所需

Git

用于克隆代码仓库

API 密钥（可选）

用于云端 LLM 模型（OpenAI、Anthropic 等）

提示：HomeClaw 可以仅使用本地模型运行。云端 API 密钥是 Mix Mode 的可选配置。

推荐模型

HomeClaw 使用本地 GGUF 模型。只需主聊天模型即可运行 HomeClaw。其他模型为可选，用于增强特定功能。

必需：主聊天模型 — 这是您开始使用所需的唯一模型。以下其他模型均为可选增强功能。

主聊天模型（必需）

主模型负责对话和工具调用。选择其中一个：

主模型（支持视觉）

Qwen3VL-4B-Instruct-Q4_K_M.gguf
mmproj-Qwen3VL-4B-Instruct-F16.gguf

备选方案：

  • Qwen3VL-2B-Instruct-Q4_K_M.gguf — 更轻量、更快、更少显存
  • Qwen3-VL-8B-Instruct-Q4_K_M.gguf — 更强大，需要更多显存
  • gpt-oss-20b-Q4_K_M.gguf — 纯文本模型，较大

嵌入模型（可选）

用于记忆、RAG 和语义搜索。没有此模型，记忆功能将受限：

嵌入模型

Qwen3-Embedding-0.6B-Q8_0.gguf

视觉模型（可选）

用于图像理解。上面的 Qwen3VL 模型已内置视觉支持，只有在使用纯文本主模型时才需要单独的视觉模型：

视觉模型（独立）

Qwen3VL-2B-Instruct-Q4_K_M.gguf
mmproj-Qwen3VL-2B-Instruct-Q8_0.gguf

工具选择模型（可选）

专门针对函数调用微调，提高工具调用准确性：

工具选择模型

Qwen3-4B-Function-Calling-Pro.gguf
模型下载：Hugging Face — 搜索 GGUF 版本。将模型放入 models/ 文件夹。

步骤 1: 安装 HomeClaw

克隆代码仓库并运行适合您平台的安装脚本。

克隆代码仓库

克隆 HomeClaw

git clone https://github.com/allenpeng0705/HomeClaw.git
cd HomeClaw

运行安装脚本

选择适合您操作系统的脚本：

Mac / Linux
Windows

在 Mac/Linux 上安装

chmod +x install.sh
./install.sh

该脚本将：

  • 安装 Python 依赖
  • 下载 llama.cpp 二进制文件
  • 下载默认模型（可选）
  • 创建默认配置文件

在 Windows 上安装（PowerShell）

.\install.ps1

或使用命令提示符

install.bat

→ 详细安装指南

步骤 2: 运行 Core

启动 HomeClaw Core 服务器以开始使用您的 AI 助手。

启动 Core

启动 HomeClaw Core

python -m main start

启动后发生什么

  • Core 在默认端口 9000 上启动
  • WebChat 可在 http://127.0.0.1:9000 访问
  • API 端点已准备好供伴侣应用使用
  • 默认模型配置从 config/llm.yml 加载

验证 Core 正在运行

打开浏览器并访问：

WebChat URL

http://127.0.0.1:9000

配置模型（可选）

编辑 config/llm.yml 来配置您的模型：

  • 本地模型：将 GGUF 文件放入 models/ 文件夹
  • 云端模型：添加 OpenAI、Anthropic 等的 API 密钥
  • Mix Mode：启用本地和云端之间的自动路由

→ 如何配置模型

步骤 3: 伴侣应用

下载并配置伴侣应用以获得最佳的 HomeClaw 体验。

下载伴侣应用

获取适合您平台的应用：

iPhone

App Store（即将推出）或从源码构建

Android

从 GitHub Releases 下载 APK

Mac

从 GitHub Releases 下载 .dmg

Windows

从 GitHub Releases 下载 .exe

连接到 Core

  1. 打开伴侣应用
  2. 进入 设置
  3. 输入 Core URL：http://127.0.0.1:9000（本地连接）
  4. 如果启用了身份验证，输入您的 API 密钥
  5. 点击 连接

功能特性

  • 聊天：与您的 AI 助手对话
  • 好友：AI 好友和家庭成员
  • 管理 Core：编辑配置文件
  • 技能：从 ClawHub 安装技能

步骤 5: 远程访问

设置远程访问以便从任何地方使用 HomeClaw。

启用身份验证

首先，使用 API 密钥身份验证保护您的 Core：

在 config/core.yml 中启用认证

auth_enable: true
auth_api_key: your-secure-api-key-here

选择隧道方法

Pinggy（推荐）

内置支持，可扫描二维码

在 core.yml 中配置

pinggy:
  token: your-pinggy-token
  enabled: true

Cloudflare Tunnel

免费、可靠、无需账户

启动隧道

cloudflared tunnel --url http://localhost:9000

Tailscale

私有网状 VPN，像本地网络一样工作

安装并连接

tailscale up

远程连接伴侣应用

  1. 启动您的隧道服务
  2. 获取公共 URL（例如 https://xxx.trycloudflare.com）
  3. 在伴侣应用中，输入此 URL 作为 Core URL
  4. 输入您的 API 密钥
  5. 连接并从任何地方聊天

→ 远程访问指南

步骤 6: Cursor 和 Claude Code

从手机使用 Cursor IDE 和 Claude Code 进行远程编程。

前置要求

  • Cursor IDE 安装在您的开发机器上
  • Claude Code CLI 已安装（npm install -g @anthropic-ai/claude-code）
  • API 密钥：Cursor API 密钥或 Anthropic/Minimax 密钥

启动 Cursor Bridge

在开发机器上启动桥接

python -m external_plugins.cursor_bridge.server

在 user.yml 中添加好友

添加 Cursor 和 ClaudeCode 好友

users:
  - id: yourname
    friends:
      - name: Cursor
        preset: cursor
      - name: ClaudeCode
        preset: claudecode

从伴侣应用使用

  1. 打开伴侣应用
  2. 选择 Cursor 或 ClaudeCode 好友
  3. 说"在 Cursor 中打开 /path/to/project"
  4. 发送编程任务如"为认证模块添加单元测试"
  5. 查看返回到您手机的结果
提示：在手机上使用前，请先设置远程访问，以便您的手机可以访问 Core，并确保 Core 可以访问您开发机器上的桥接。

→ Cursor 和 Claude Code 指南
