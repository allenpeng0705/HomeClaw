ClawHub 技能

从 ClawHub 注册表扩展 HomeClaw 技能。

概述

ClawHub (clawhub.ai) 是 OpenClaw 技能的公共注册表。它托管版本化的技能包，包含元数据、标签和发现功能。

什么是技能？

技能是可重用的工作流，用于扩展 AI 助手的功能。每个技能是一个文件夹，包含：

  • SKILL.md — 描述、说明和配置（YAML 前言 + Markdown 正文）
  • scripts/ — 可选的 Python、Shell 或 JavaScript 脚本
  • references/ — 可选的文档和示例

可用技能

ClawHub 上可用的技能示例：

  • maton-api-gateway — 连接 100+ 网络服务（Gmail、Google Calendar、Slack、Notion、Salesforce、HubSpot 等）的统一 API
  • summarize — 总结文档和文章
  • weather — 获取天气信息
  • html-slides — 创建 HTML 演示文稿
  • ppt-generation — 生成 PowerPoint 幻灯片
  • image-generation — 使用 AI 生成图像
  • linkedin-writer — 撰写 LinkedIn 帖子
  • 更多...
推荐： maton-api-gateway 技能对于将 HomeClaw 连接到 100+ 流行网络服务至关重要，包括 Gmail、Google Workspace、Slack、Notion、Salesforce、HubSpot、Stripe 等。它为所有这些服务提供统一的 API 接口。

工作原理

  1. 搜索 — 在 ClawHub 上按查询查找技能
  2. 下载 — 将 OpenClaw 技能下载到暂存文件夹
  3. 转换 — 转换为 HomeClaw 格式
  4. 加载 — Core 自动加载技能

设置 ClawHub 令牌

要在不使用浏览器认证的情况下使用 ClawHub，请在配置中设置您的令牌。

获取您的令牌

  1. 访问 clawhub.ai
  2. 登录或创建账户
  3. 从账户设置获取 API 令牌

在 core.yml 中配置

将您的 ClawHub 令牌添加到 config/core.yml：

配置 ClawHub 令牌

clawhub_token: "clh_YourTokenHere"

配置目录

HomeClaw 使用以下目录存放技能：

core.yml 中的技能目录

skills_dir: skills
external_skills_dir: external_skills
clawhub_download_dir: downloads
skills_extra_dirs: []
目录 用途 默认值
skills_dir 内置技能（受保护） skills/
external_skills_dir 转换后的 ClawHub 技能 external_skills/
clawhub_download_dir 下载暂存 downloads/
skills_extra_dirs 额外的技能文件夹 []

搜索技能

使用伴侣应用、Portal 或 CLI 在 ClawHub 上搜索技能。

使用伴侣应用

  1. 打开伴侣应用
  2. 进入 设置 → 技能
  3. 点击 搜索技能
  4. 输入搜索查询
  5. 浏览结果并选择要安装的技能

使用 Portal

  1. 启动 Portal：python -m main portal
  2. 打开 http://127.0.0.1:18472
  3. 导航到 技能 部分
  4. 使用搜索框查找技能
  5. 点击所需技能上的 安装

使用 CLI

通过 CLI 搜索技能

python -m main skills search "weather"

带限制搜索

python -m main skills search "slides" --limit 10

安装技能

从 ClawHub 安装技能以扩展助手的功能。

安装流程

  1. 下载 — 技能被下载到 downloads/ 文件夹
  2. 转换 — OpenClaw 格式转换为 HomeClaw 格式
  3. 安装 — 转换后的技能放置在 external_skills/
  4. 加载 — Core 在下次请求时自动加载技能

使用伴侣应用

  1. 搜索技能（见上文）
  2. 点击技能上的 安装
  3. 等待下载和转换
  4. 技能即可使用

使用 Portal

  1. 在 Portal 中搜索技能
  2. 点击 安装
  3. 如需要指定版本（例如 summarize@1.0.0）
  4. 等待安装完成

使用 CLI

通过 CLI 安装技能

python -m main skills install summarize

安装特定版本

python -m main skills install summarize@1.0.0
注意： 安装的技能会放入 external_skills/ 文件夹。skills/ 中的内置技能受保护，无法通过 API 删除。

技能文件夹

HomeClaw 按特定顺序从多个目录加载技能。

加载顺序

技能按以下顺序加载（首次匹配优先）：

  1. skills_dir — 内置技能（skills/）
  2. external_skills_dir — ClawHub 安装的技能（external_skills/）
  3. skills_extra_dirs — 额外的自定义文件夹

内置技能 (skills/)

受保护的文件夹，包含 HomeClaw 附带的核心技能：

  • html-slides — 创建 HTML 演示文稿
  • ppt-generation — 生成 PowerPoint 幻灯片
  • image-generation — 使用 AI 生成图像
  • weather — 天气信息
  • summarize — 文档总结
  • 更多...

外部技能 (external_skills/)

从 ClawHub 安装的技能文件夹：

  • 技能自动转换为 HomeClaw 格式
  • 可通过伴侣应用、Portal 或 CLI 删除
  • 稳定后，可移动到 skills/ 永久使用

自定义技能文件夹

在 core.yml 中添加额外的技能文件夹：

添加自定义技能文件夹

skills_extra_dirs:
  - /path/to/custom/skills
  - ./my-skills

转换过程

OpenClaw 技能在安装过程中自动转换为 HomeClaw 格式。

转换内容

  • SKILL.md — 保留并适配
  • scripts/ — Python、Shell、JavaScript 文件
  • references/ — 文档和示例
  • skill.yaml — 合并到 SKILL.md 前言

支持的脚本类型

扩展名 运行时 说明
.py, .pyw Python Core 的 Python 解释器
.sh, .bash Shell Bash shell
.js, .mjs, .cjs Node.js 需要安装 Node.js
.ts TypeScript 需要 tsx 或 ts-node

手动转换

您也可以手动转换 OpenClaw 技能文件夹：

手动转换技能

python scripts/convert_openclaw_skill.py /path/to/openclaw/skill external_skills/skill-name

管理技能

通过各种界面查看、启用、禁用和删除技能。

列出已安装的技能

通过 CLI 列出技能

python -m main skills list

禁用技能

将技能文件夹名称添加到 core.yml 的禁用列表：

禁用技能

skills_disabled:
  - skill-name-1.0.0
  - another-skill

删除技能

只能删除 external_skills/ 中的技能：

通过 CLI 删除技能

python -m main skills remove skill-name

或通过伴侣应用 / Portal：

  1. 进入技能部分
  2. 找到技能
  3. 点击 删除
警告： skills/ 中的内置技能无法删除。请使用 skills_disabled 来禁用它们。
