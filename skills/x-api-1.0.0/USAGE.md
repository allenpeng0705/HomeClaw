# How to ask (X API skill)

- "Post a tweet: Hello world"
- "Tweet this: [your text]"
- "Post to X: [your text]"
- "What are my latest tweets?"
- "Read my X timeline" / "Show my last 10 tweets"

Set `X_ACCESS_TOKEN` (or `x_access_token` in this skill's config.yml). Get the token at [developer.x.com](https://developer.x.com) after creating an app and completing the OAuth flow.
